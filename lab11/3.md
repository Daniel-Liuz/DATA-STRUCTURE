# 最大子数组和
# Description

给你一个整数数组nums，请你找出一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。

子数组是数组中的一个连续部分。


# Input

第一行输入一串个整数，表示nums数组，用空格分隔。


# Output

输出一个整数，表示最大和


# Sample Input 1

9

-2 1 -3 4 -1 2 1 -5 4

# Sample Output 1

6
# Sample Input 2

1
# Sample Output 2

1
# Sample Input 3

5

5 4 -1 7 8

# Sample Output 3

23
